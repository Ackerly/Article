##父子组件间的传值
1.使用Prop和Emit  
父组件  
html部分
```
<template>
    <A :title="msg"></A>
</template>
```
js部分
```
import A from "./A.vue";

export default {
    name: 'Parent',
    components: {
        A,
    },
}
```

A组件  
html部分
```
 <div>A{{title}}</div>
```
js部分
```
export default {
    name: "a",
    components: {
        B
    },
    props: {
        title: String
    }

}
```
